　Turbo C 2.0/Turbo C++ 1.0 用の GLIO ライブラリです。


□　使用法

　プログラムには、必ず glio.h をインクルードしてください。
　すべての関数はラージモデルで作成されていますが、glio.h を使用する限り
タイニーモデルを除くすべてのメモリモデルでも使用できます。

例）tcc prog.c glio.lib

　関数の詳細については、GLIO.DOC を参照してください。


□　ソースについて

　プログラムは、gprintf 関係を除きTurbo AssemblerのIDEAL モードで
記述されています。付属の makefile を使うか、コマンドラインで /mx
オプションをつけてアセンブルしてください。
　gprintf.c (gprintf, vgprintf) は、書式つきでグラフィックに文字表示す
るための関数のサンプルです。これらの関数を使用するプログラムと一緒に
コンパイル／リンクするなどで使用してください。
　gprintfは、Turbo C 2.0/Turbo C++ 1.0の内部関数を利用していますが、
両者の間に若干の違いがあるため、それぞれの版で利用するためには
コンパイルしなおさなければなりません。


□その他

　Ｃマガジン９０年２月号に付属のものは、動作がかなり不安定でしたが、
　その点については改善されています。また、グラフィック画面を
　ファイルにセーブしたり、ファイルからロードする関数(gsave, gload)が
　追加されています。
